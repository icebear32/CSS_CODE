相对定位 relative

相对定位是元素在移动位置的时候，是相对于它原来的位置来说的

语法

选择器 { position: relative; }

    相对定位的特点
  • 相对于原来的位置来移动的
  • 原来在标准流的位置继续占有，后面的盒子仍然以标准流的方式对待
  • 相对定位并没有脱标
