类选择器

想要差异化选择不同的标签，单独选一个或者某几个标签，可以使用类选择器
类选择器在 HTML 中以 class 属性表示，在 CSS 中，类选择器以一个点“.”号显示

语法

例如，将所有拥有 red 类的 HTML 元素均为红色

.类名{
        属性1: 属性值1;
        属性2: 属性值2;
        属性3: 属性值3;
        ...
}

注意

  • 使用“.”（英文点号）进行标识，后面紧跟类名
  • 给这个标签起另一个名字
  • 可使用中横线命名
  • 不要使用纯数字、中文等命名，尽量使用英文字母来表示
  • 命名最好见名知意
